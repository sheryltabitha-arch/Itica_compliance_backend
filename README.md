Itica Compliance Backend is a FastAPI-based backend that powers the Itica platform.

Itica helps compliance teams manage regulatory obligations, reduce false positives, track regulatory updates, and streamline compliance workflows across financial institutions, healthcare organizations, and enterprises.

The platform is designed to act as the central infrastructure for compliance teams, helping organizations reduce time spent interpreting regulations and lower the cost of regulatory compliance.

## Features

- Regulatory obligation tracking
- Compliance workflow management
- Reduction of regulatory false positives
- Centralized compliance documentation
- Real-time regulatory monitoring
- Scalable API infrastructure
## Installation

Clone the repository:

git clone https://github.com/sheryltabitha-arch/Itica_compliance_backend

cd Itica_compliance_backend

Install dependencies:

pip install -r requirements.txt

Run the server:

uvicorn main:app --reload
